# qBittorrent Assistant

Edge 扩展，通过扩展弹出窗口管理qBittorrent任务，快速添加磁力。

## 功能
- 多服务器管理
- 实时下载监控
- 磁力链接一键添加
- 速度限制控制
- 自动状态刷新
- 下载完成通知

## 安装
1. Edge扩展商店搜索"qBittorrent磁力助手"安装
2. 或手动加载扩展文件夹

## 使用
1. 配置 qBittorrent 服务器地址
2. 右键磁力链接发送到 qBittorrent
3. 点击扩展图标监控状态

## 配置
- 活动刷新间隔：5秒 (默认)
- 空闲刷新间隔：关闭 (默认)
- 支持切换顺序下载
- 开关备用速度限制
- 可设置默认分类和路径
